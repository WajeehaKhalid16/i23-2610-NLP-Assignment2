# i23-2610-NLP-Assignment2
# NLP Assignment 2 Neural NLP Pipeline

## Overview

This project is a complete NLP pipeline built from scratch using PyTorch. The goal of this assignment was to understand how different NLP components work internally instead of relying on pretrained libraries.

The pipeline includes TF IDF and PPMI representations, a Skip gram Word2Vec model, a BiLSTM model for POS tagging and Named Entity Recognition, and a Transformer encoder for topic classification.

All parts were implemented manually without using pretrained models or built in transformer modules.

## How to Run

You can run the project either using the notebook or the Python scripts.

Run notebook
Open notebook assignment2.ipynb and run all cells in order. Make sure every cell produces output.

Run Part 1 Word Embeddings
python src/part1_embeddings.py

This will generate TF IDF matrix, PPMI matrix, and Word2Vec embeddings and save them in the embeddings folder.

Run Part 2 BiLSTM for POS and NER
python src/part2_bilstm.py

This will train the POS and NER models and save them in the models folder.

Run Part 3 Transformer Encoder
python src/part3_transformer.py

This will train the classification model and save it in the models folder.

## Notes
